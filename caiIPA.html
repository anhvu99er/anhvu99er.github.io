Hướng dẫn cài Ipa bằng ESign

Bài viết này dành cho các bạn mới sử dụng ESign, làm thế nào để ký IPA và cài lên thiết bị của mình

Bước 1: Người bán sẽ cũng cấp cho bạn hai đường link, hãy bấm cài cả hai link đấy.
1. Link ESign trực tiếp
2. Link tải file chứng chỉ

Sau khi cài đặt ESign hoàn tất hãy mở nó lên ( Nếu máy báo yêu cầu chế độ nhà phát triển). Vào cài đặt máy - Quyền riêng tư và bảo mật - Chế độ nhà phát triển - Bật nó lên.



Bước 2: Mở Esign - bấm ba chấm - nhập.

Tìm file chứng chỉ các bạn tải về ở bước 1 và bấm cho nó tự động nhập vào ESign như thế này.



Bước 3: Giải nén file chứng chỉ vừa nhập vào. Thường sẽ có ba file:
1. File cert.p12
2. File cert.mobileprovison
3. Pass chứng chỉ



Bước 4: Tiến hành nhập chứng chỉ.
Chọn file cert.p12 - nhập kho chứng chỉ.
Chọn file cert.mobileprovision - nhập.



Vậy là xong chứng chỉ của bạn đã nhập thành công. Nếu cần hãy xem cách cài đặt IPA tại đây
